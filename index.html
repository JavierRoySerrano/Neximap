📖 Quick Actions

Basic Workflow

  1. Create nodes by double-clicking on the canvas or selecting a shape from the toolbar
  2. Connect nodes by clicking and dragging from one node to another
  3. Edit properties using the Inspector panel on the right (press I to toggle)
  4. Organize your diagram with containers and groups
  5. Save your work with Ctrl+S or File → Save

Canvas Navigation

  • Pan: Click and drag on empty canvas space
  • Zoom: Ctrl + mouse wheel, or use the zoom controls in the toolbar
  • Fit to view: Press Enter or click the Fit button

💡 Tips & Tricks

Pro tips to boost your productivity with NexiMap Studio.

  • ⛡
    Quick Node Creation
    Double-click anywhere on the canvas to create a new node instantly. The node will use your currently selected shape tool.
  • 🎨
    Format Painter
    Use the brush tool to copy formatting from one node and apply it to others. Great for consistent styling.
  • 📐
    Snap to Grid
    Enable the grid for precise alignment. The grid size can be customized from Fine (10px) to Extra Large (100px).
  • 🔍
    Multi-Select with Alt+Drag
    Hold Alt and drag to create a selection box. All items within the box will be selected for bulk operations.
  • 🏷️
    Power of Tags
    Tag items consistently to enable powerful filtering. Use the Label Manager to batch-add or remove tags across multiple items.
  • 💾
    Auto-Save Recovery
    NexiMap Studio automatically saves your work. If something goes wrong, use File → Recovery → Recover Autosave.
  • 🗺️
    GPS Coordinates
    Add latitude and longitude to nodes to enable geographic visualization. Click the 🌐 button on the map to sync schematic positions from GPS.
  • 📊
    Data Table Power
    Open the Data Table (Ctrl+T) for spreadsheet-like editing. Click column headers to sort, use the search to filter, and double-click cells to edit.

❓ Frequently Asked Questions

How do I connect two nodes? +
Click on the source node and drag towards the target node. A link will be created automatically when you release the mouse over another node.
How do I delete items? +
Select the item(s) you want to delete and press the Delete or Backspace key. You can also click the trash icon in the toolbar. Use Ctrl+Z to undo if needed.
Can I import data from Excel? +
Yes! Go to File → Import/Export → Download Import Template to get the Excel template. Fill in your data and use Import from Spreadsheet to load it.
How do I show my network on a map? +
First, add GPS coordinates (latitude/longitude) to your nodes in the Inspector panel. Then click the "Geo" button for a static map or the "Map" button for an interactive map.
How do I export my diagram as an image? +
Go to File → Import/Export and choose Export as SVG, PNG, or JPG. SVG is best for scaling, while PNG/JPG are better for sharing.
How do I find the shortest path between nodes? +
Click the "Routes" button in the toolbar to open the Route Finder. Select your start and end nodes, choose your optimization criteria (cost, latency, hops), and click Find Routes.
How do I save my file format preferences? +
NexiMap Studio saves your preferences automatically, including grid settings, zoom level, and panel positions. Your financial settings and default styles are also preserved.
